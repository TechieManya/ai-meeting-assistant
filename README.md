# AI Meeting Summarizer

An AI-powered meeting assistant that automatically records meetings, converts speech to text, generates concise summaries, extracts action items, and sends email reports to participants.

## Project Goals

- Record meeting audio
- Convert speech to text
- Identify speakers (speaker diarization)
- Generate AI-powered summaries
- Extract action items and decisions
- Send automated email reports
- Provide a dashboard for meeting insights

## Tech Stack

### Frontend
- React
- Vite

### Backend
- FastAPI
- Python

### AI & APIs
- Deepgram (Speech-to-Text)
- Gemini API (Summaries & Action Items)
